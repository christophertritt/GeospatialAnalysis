# Green Stormwater Infrastructure Data Search Results
**Washington State Cities: King and Pierce Counties**

*Search Date: December 3, 2025*

---

## Executive Summary

A comprehensive search was conducted across city, county, and state open data portals to identify publicly available green stormwater infrastructure (GSI) and permeable pavement datasets for:

**King County Cities:** Tukwila, Renton, Kent, Auburn  
**Pierce County Cities:** Sumner, Puyallup, Fife, Tacoma

### Key Findings:

**Limited Public GSI Datasets Available** - Most cities do not maintain publicly accessible, downloadable GSI-specific datasets on their open data portals. However, Tacoma has documented GSI programs and permeable pavement projects.

## Data Quality Considerations

When obtaining GSI data, request the following attributes:

### Essential Fields:
- **Location:** Latitude/Longitude or address
- **Facility Type:** Rain garden, bioretention, permeable pavement, bioswale, etc.
- **Installation Date:** Year installed
- **Size/Capacity:** Square footage, volume, drainage area served
- **Status:** Active, inactive, maintenance needed
- **Owner:** Public, private, partnership

### Desirable Fields:
- **Design specifications:** Soil type, vegetation, infiltration rate
- **Maintenance history:** Frequency, type of maintenance
- **Performance data:** Monitoring results if available
- **Cost information:** Installation and maintenance costs
- **Photos/Documentation:** Visual records

## Conclusion

**Current State:**
- **Limited public GSI datasets** available for direct download
- **Tacoma** has the most comprehensive public documentation of GSI programs
- Most other cities likely maintain GSI data internally but not on public portals

**Data Acquisition Strategy:**
1. Start with Tacoma as proof-of-concept
2. Submit targeted public records requests to all other cities
3. Contact county agencies for regional data
4. Plan for potential field verification and supplementary data collection

**Expected Timeline:**
- Tacoma data request: 1-2 weeks
- Public records requests: 2-4 weeks per city
- Complete data compilation: 2-3 months

**Alternative Approach:**
If comprehensive GSI datasets cannot be obtained, consider expanding to broader "stormwater facilities" datasets and filtering for green infrastructure characteristics based on facility type designations.
